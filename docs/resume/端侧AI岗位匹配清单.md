\## 2. 不建议优先投递的岗位



\### 纯嵌入式 C/C++ 开发



暂时不优先。



原因：



\* 你目前 C/C++ 不强

\* 项目主线是 Python + PyTorch + ONNX Runtime

\* 纯嵌入式岗位更看重 C、C++、RTOS、驱动、MCU、Linux 内核等能力



\---



\### 纯硬件测试 / 通信网优



不建议作为主线。



原因：



\* 和你现在的 AI 部署、边缘智能方向不完全一致

\* 容易把职业路线带回传统电子信息岗位

\* 上限和成长路径不如 AI 工程化方向清晰



\---



\### 纯大模型算法研究岗



短期不作为主攻。



原因：



\* 这类岗位通常更看重顶会论文、深度学习算法研究能力、大模型训练经验

\* 你当前更适合从端侧 AI、模型部署、边缘智能切入
